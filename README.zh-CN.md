## vitepress-markmap-preview ![npm](https://img.shields.io/npm/v/vitepress-markmap-preview.svg)

- 📝 支持列表、表格、图片等多种 Markdown 元素
- 🎨 可定制主题和布局
- 🔧 与 VitePress 无缝集成
- 📁 组件标签支持文件预览
